# LEGO Robotics Team Website

A React-based website for a LEGO-themed robotics team featuring three main pages: Home, About, and Robotics.

## Features

- LEGO-themed design with brick-inspired UI elements
- Responsive layout that works on mobile and desktop
- React Router for navigation between pages
- Styled Components for component-based styling

## Pages

1. **Home** - Welcome page with team highlights
2. **About** - Information about the team, mission, and members
3. **Robotics** - Details about robotics projects and competitions

## Getting Started

### Prerequisites

- Node.js (v14 or later)
- npm or yarn

### Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/lego-robotics-team.git
   cd lego-robotics-team
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Start the development server:
   ```
   npm start
   ```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Building for Production

To build the app for production:

```
npm run build
```

This creates an optimized production build in the `build` folder.
