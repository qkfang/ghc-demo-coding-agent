# Lego Robotics Team Website

A React-based website for our Lego Robotics Team, showcasing our projects, team information, and robotics activities.

## Features

- **Home Page**: Welcome page with team highlights and mission
- **About Page**: Information about our team, mission, and core values
- **Robotics Page**: Details about our projects, competitions, and skills development

## Technology Stack

- React 18
- React Router 6 for navigation
- Modern CSS with responsive design
- Emoji icons for visual appeal

## Getting Started

### Prerequisites
- Node.js (version 14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd gh-coding-agent
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

The application will open in your browser at `http://localhost:3000`.

## Available Scripts

## Pages

### Home
Welcome page featuring:
- Team introduction and mission
- Highlight cards showing Championships, Innovation, and Teamwork
- Engaging hero section with team motto

### About
Team information including:
- Mission statement
- Team statistics (size, founding year, school, achievements)
- Core values (Innovation, Collaboration, Learning, Fun)

### Robotics
Technical content featuring:
- Current robot projects with descriptions
- Competition information
- Skills development areas (Programming, Engineering, Problem Solving)
